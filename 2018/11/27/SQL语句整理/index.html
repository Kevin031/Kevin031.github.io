發表於 2018-11-27 | 分類於 后端

一些最重要的SQL命令

SELECT - 从数据库中提取数据
UPDATE - 更新数据库中的数据
DELETE - 从数据库中删除数据
INSERT INTO - 向数据库中插入新数据
CREATE DATABASE - 创建新数据库
ALTER DATABASE - 修改数据库
CREATE TABLE - 创建新表
ALTER TABLE - 变更（改变）数据库表
DROP TABLE - 删除表
CREATE INDEX - 创建索引
DROP INDEX - 删除索引

模糊查询

ps:

% - 表示多个字值，_下划线表示一个字符
M% - 通配符，正则表达式，表示的意思是模糊查询信息为M开头的
%M% - 表示查询包含M的所有内容
%M_ - 表示查询M在倒数第二位的所有内容

  1. 不带比较运算符的WHERE子句
