ES6新增Promise对象速查文档

發表於 2017-11-28 | 分類於 前端

Promise是ES6中引入的对象，它解决了回调的以下痛点

1.回调过早

Promise自动解决，被then调用

2.次数过多

Promise内部机制决定了回调为单向不可逆

3.报错被吞掉

Promise状态被拒绝时reject(error)能接收

4.回调没有被调用

用race方法能解决

Promise的本质

这是一个可以包含异步操作的对象

Primise的状态

Fulfilled：成功
Rejected：拒绝
Pending：进行中

状态的变化

每一个操作都有resolve和reject两种反应，导致不同的结果

1.resolve -> pending => Fulfilled
2.reject -> Pending => Rejected

链式

).then(
// 第二个异步操作，接收上一个操作的返回值
).then(
// 第三个异步操作，接收上一个操作的返回值
