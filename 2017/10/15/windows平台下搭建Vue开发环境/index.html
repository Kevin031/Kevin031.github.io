webpack基本配置

發表於 2017-10-15 | 分類於 前端

1.安装node.js

(安装cnpm)

npm install -g cnpm --registry=http://registry.npm.taobao,org

不过用cnpm的话有时候也会出现依赖包不齐的情况，我是优先选择npm的

3.安装vue-cli脚手架构建工具

npm install -g vue-cli

4.初始化一个vue项目

vue init webpack firstVue

其中 firstVue 是项目文件夹的名称

5.安装项目依赖包

npm install
